How do I get a substring from within a string?
    To get a substring, use the substr function using 2 parameters. The first parameter is the 
    starting index, and the second parameter is how many characters you want the substring to be.

How do I create an escape character in C++
    To create an escape character, use a back slash.

How do I convert a string to another data type?
    To convert a string to another data type, you can use functions such as stoi and stof.

What is a raw string and when would I use it?
    A raw string is a string without escape characters and you can use it if you need to print
    certain escape character.
